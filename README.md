# Edufine Money Docs Make

학교 품의서 작성을 돕는 Windows용 견적서 자동 분석 프로그램입니다.

## 다운로드

최신 Windows 실행 파일은 GitHub Releases에서 받을 수 있습니다.

- https://github.com/ryowoo2726-hue/edufine_money_docsmake/releases/latest

다운로드한 `AutoMoneyDocMake-windows-x64.zip`을 압축 해제한 뒤 `flutter_ui.exe`를 실행합니다.

## 기능

- PDF/이미지 견적서 업로드
- Gemini API 기반 품목 추출
- 사이트별 견적서 인식 규칙 적용
- 품의용 Excel 파일 생성
